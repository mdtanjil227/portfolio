# Personal Portfolio Website

This is my personal developer portfolio website built with **HTML, CSS, and JavaScript**.
It showcases my skills, projects, and provides a way for visitors or recruiters to contact me directly.

The website is designed to be **clean, responsive, and lightweight**, focusing on simplicity and usability.

---

## Live Website

The portfolio is available at:

```
https://mdtanjil227.github.io/portfolio
```

---

## Features

* Responsive layout for desktop, tablet, and mobile
* Skills section highlighting technologies I work with
* Projects section to showcase development work
* Contact form integrated with **EmailJS** to send messages directly to my email
* Smooth scrolling navigation
* Clean and minimal UI design

---

## Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **Font Awesome**
* **Google Fonts (Poppins)**
* **EmailJS** for contact form functionality

## Contact Form

The contact form uses **EmailJS** to allow visitors to send messages directly from the website without needing a backend server.

Users can submit:

* Name
* Email
* Message

Messages are delivered directly to my email inbox.
